# Web Dev & Accessibility — Project Milestone #3


## Overview

This is a small static website created for Intro to Web Development (IT 3203) with Prof. Denise Tucker at KSU. The site explores web accessibility through semantic HTML, forms, ARIA labelling, color & contrast, and a quiz for self-assessment.

## New Features and Updates
- Accessibility features (back-to-top button, escape key for navigation, simplistic color scheme, user-friendly forms, reduced motion)
- Sidebar navigation 
- A quiz with per-question feedback, final scoring, and option to retry 
- Responsive layout
- Site map page
- Improved README document
